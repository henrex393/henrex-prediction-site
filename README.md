# Henrex Football Prediction Site

Henrex Predictions is a football match prediction platform that provides data-driven insights for football fans and bettors. The website shows live matches, major league fixtures, and helpful betting predictions.

## Features

- Live football matches
- Major league match listings
- Match result predictions
- Over/Under goals predictions
- BTTS (Both Teams To Score) predictions
- Corner predictions
- Hot match highlights
- Affiliate betting links

## Technologies Used

- Python
- Flask
- HTML
- CSS
- API-Football

## Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open your browser and visit:

http://127.0.0.1:5000
